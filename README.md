# LocalBox Miner

LocalBox Miner is a React-based offline data management application that allows viewing of data in a Dashboard (Home),
Adding Records, Editing Records, and Deleting Records with Confirmation.

## Preparation

First, make sure to run ```npm install``` and then run ```npm run dev```, then type ```o``` character on the terminal
to open it in browser.
